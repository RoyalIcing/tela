# Tela

Alpha version can be loaded via jsdelivr: <https://cdn.jsdelivr.net/gh/RoyalIcing/tela@795ffd8719b5614bd22e0f41318b79b8df43c104/tela.css>

## Layout

### X

```html
<ul x-x>
  <li>First
  <li>Second
  <li>Third
</ul>
```

- Uses flexbox with row direction.

### Y

```html
<ul y-y>
  <li>First
  <li>Second
  <li>Third
</ul>
```

- Uses flexbox with column direction.
